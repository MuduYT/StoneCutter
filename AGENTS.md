# StoneCutter Agent Instructions

Codex agents working in this repository must follow the project rules in:

- `.codex/rules/stonecutter.codexrules.md`

The rules are the Codex-formatted version of the original Windsurf workflow rules from `.windsurf/workflows/rules.md`. Keep the Codex rules as the source used by Codex, and only update the Windsurf rules separately when editor-specific behavior is needed.

When making implementation or behavior changes, keep supporting files current as part of the same task. This includes `CHANGELOG.md`, README/docs, and agent or editor rules when they are affected.
